# CLAUDE.md — {项目名}

极简项目协同工具。以周为单位，每次新建文件，旧文件保留。

## 每周节奏

**周四/周五前 — 各人写周报：**
在 `10-item/{name}/` 下新建 `YYMMDD.md`

**周五 — PM 准备会议材料：**
1. 用 AI 从各人周报"卡在哪"汇总问题 → 对比上周融合列表 → 写入 `03-meeting/YYMMDD.md`
2. 同步更新 `00-proj/YYMMDD.md`

**周五 — 周会：**
1. 打开 `03-meeting/YYMMDD.md`，逐模块过进度
2. 融合问题列表逐条决议（解/搁置/调目标）
3. 方向检查 + 遗留事项确认
4. 会中边讨论边更新 — 各人改周报，PM 改会议材料和项目状态
5. 已解决的问题归档到 `01-artifacts/问题归档.md`

**散会标准：** 各人已更新周报 · PM 已更新项目状态 · 卡点全有决议 · 方向统一 · 暂不做的事有原因有重启时间

## 目录结构

```
00-proj/              ← 项目状态（PM 每周新建）
01-artifacts/         ← 产物
02-sync/              ← 对接记录
03-meeting/           ← 会议材料
04-问题/              ← 问题升级追踪
10-item/{name}/       ← 各人每周文件
99-current/           ← 本项目模板
99-tool/              ← 通用工具箱
```
