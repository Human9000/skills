# 接口约定

> 任意两人工作有交接面时，开工前花 10 分钟填完。双方确认签字。

## 模板

```
上游（产出方）：[谁]
下游（使用方）：[谁]
交付什么：[具体文件/数据/格式]
格式/规范：[能多具体就多具体]
交付时间：[日期]
验证方式：[下游怎么确认交付物正确]
如果延期：[下游可以先做什么 / 谁被通知]
```

## 什么时候必须写

- 工作有**时间先后**（A 做完 B 才能开始）
- 工作**可并行但需接合**（A 的输出格式决定 B 怎么读）
- 其中一人是**兼职**
- 其中一人是**新人**

## 变更规则

- 接口变更 → 上游**主动推送**通知下游（私聊/@），不放群公告
- 涉及兼职成员 → 24h 内未确认 → 电话确认
- 下游确认后更新文件
